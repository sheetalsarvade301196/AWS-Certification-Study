# Day 1 Quiz Answers & Notes

Q1. When would you choose EFS over EBS?
A1. When you need a shared, POSIX-compliant filesystem accessible by multiple EC2 instances across AZs (e.g., shared web content).

Q2. What does 'versioning' provide in S3?
A2. Ability to preserve, retrieve, and restore every version of every object stored in an S3 bucket.

Q3. Best practice for IAM?
A3. Least privilege, roles for services, enable MFA for human principals, avoid long-lived credentials.
